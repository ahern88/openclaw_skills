# 博客写作示例

## 示例 1：技术教程

**用户请求**:
```
帮我写一篇博客，介绍 Docker 入门
```

**AI 执行**:
1. 创建 Markdown 文件
2. 生成完整教程（安装、配置、使用）
3. hexo generate
4. hexo deploy
5. 返回链接

**输出**:
```
✅ 文章已发布！

📝 Docker 入门教程
🔗 https://ahern88.github.io/2026/03/06/docker-install-guide/
```
