# 基金股票跟踪技能

自动跟踪基金和股票持仓，每日计算收益并推送报表。

---

## 📋 功能

1. **基金净值获取** - 通过天天基金 API 获取实时净值
2. **股票价格获取** - 通过新浪财经 API 获取实时股价
3. **收益计算** - 对比前一日数据，计算盈亏
4. **报表生成** - 汇总总收益、最佳/最差表现
5. **定时推送** - 工作日晚上自动推送到飞书

---

## 🔧 配置

### 环境变量

- `FEISHU_USER_ID`: 飞书用户 ID（用于推送）

### 持仓数据存储

持仓数据保存在 `/home/admin/.openclaw/workspace/memory/fund-holdings.json`

## 📊 API 来源

| 类型 | 数据源 | API 地址 |
|------|--------|---------|
| 基金净值 | 天天基金 | http://fundgz.1234567.com.cn/js/{fund_code}.js |
| 股票价格 | 新浪财经 | http://hq.sinajs.cn/list={market}{stock_code} |

## 💡 使用示例

### 示例 1：添加基金

**用户**: 添加基金：代码 002943, 名称 广发多因子混合，金额 90000

**AI 执行**:
1. 解析基金信息
2. 保存到持仓文件
3. 获取当前净值
4. 确认添加成功

### 示例 2：每日推送

**定时任务**: 工作日 21:00

**AI 执行**:
1. 获取所有基金/股票的当日净值/股价
2. 对比前一日数据计算收益
3. 生成汇总报表
4. 推送到飞书

**推送内容**:
```

总资产：561,794.37 元
当日收益：+178.80 元
累计收益：+21,681.29 元

基金：
✅ 广发多因子混合 +343.16 元
❌ 国富全球科技 -513.93 元

股票：
✅ 贵州茅台 +1,200 元
...
```
