# 📊 基金组合优化与回测技能

基于**天天基金网实时数据**的基金组合回测工具，支持年平衡策略和现代投资组合理论优化。

## ✨ 功能特点

- 🔄 **实时数据**：直接调用天天基金 API，获取最新历史净值
- 📈 **组合回测**：支持多基金组合，按每天净值精确计算
- 🎯 **年平衡策略**：每年指定月份自动再平衡
- 📊 **风险分析**：年化收益、最大回撤、夏普比率、卡玛比率等
- 💾 **自动缓存**：数据本地缓存 7 天，避免重复请求

## 📦 安装依赖

```bash
uv pip install pandas numpy requests
```

## 🚀 使用方法

### 基础回测

```bash
python scripts/fund_backtest_tiantian.py \
  --funds 002943,159985,165531 \
  --weights 0.4,0.3,0.3 \
  --start-date 2019-01-01
```

### 指定回测期间

```bash
python scripts/fund_backtest_tiantian.py \
  --funds 002943,159985,165531,161128 \
  --weights 0.25,0.25,0.25,0.25 \
  --start-date 2019-09-24 \
  --end-date 2026-03-14
```

### 自定义再平衡月份（每年 6 月）

```bash
python scripts/fund_backtest_tiantian.py \
  --funds 002943,159985,165531 \
  --weights 0.4,0.3,0.3 \
  --rebalance-month 6
```

### 输出 JSON 格式

```bash
python scripts/fund_backtest_tiantian.py \
  --funds 002943,159985,165531 \
  --weights 0.4,0.3,0.3 \
  --json
```

## 📋 参数说明

| 参数 | 说明 | 必填 | 默认值 |
|------|------|------|--------|
| `--funds` | 基金代码列表，逗号分隔 | 是 | - |
| `--weights` | 配置比例，逗号分隔（总和=1） | 是 | - |
| `--start-date` | 回测开始日期 YYYY-MM-DD | 否 | 2019-01-01 |
| `--end-date` | 回测结束日期 YYYY-MM-DD | 否 | 今天 |
| `--rebalance-month` | 再平衡月份 1-12 | 否 | 1 |
| `--initial-capital` | 初始资金 | 否 | 1000000 |
| `--json` | 输出 JSON 格式 | 否 | False |

## 📊 输出指标

### 核心指标
- **累计收益率**：整个回测期间的总收益
- **年化收益率 (CAGR)**：年化复合增长率
- **年化波动率**：风险指标
- **最大回撤**：历史最大亏损幅度
- **夏普比率**：风险调整后收益（无风险利率 3%）
- **卡玛比率**：年化收益/最大回撤

### 年度/季度收益
- 各年度收益率和最大回撤
- 各季度收益率列表
